Welcome to my personal site!

True, this is a simple, minimal start. I make few claims, at the moment, to anything fancy here. Instead, I simply want to talk about this site itself.

This is a project in Go. I find it a pretty interesting language, just with a name that isn't ideal. I'm supposed to Google for "Go"? "Golang" helps, but my oh my, I'm just spreading the bikeshedding aren't I.

Among the reasons I find Go interesting is simply how easy it was to get this site up and running. I'm using Echo, "A fast and unfancy micro web framework for Go". Yet even with features in my server.go file that I'm not making use of, the initial server implementation took 64 lines of code. has http Method and Path logging, responds in microseconds. It's ridiculously more than I need for a trivial blog/personal project, and it's awesome.
