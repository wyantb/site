Hello World

Welcome to my personal site!

True, this is a simple, minimal start. I make few claims, at the moment, to anything fancy here. Instead, I simply want to talk about this site itself.

This is a project in Go. I find it a pretty interesting language, just with a name that isn't ideal. I'm supposed to Google for "Go"? "Golang" helps, but... The name really gets me bikeshedding about naming. Sigh. Is what it is.

Anyway, one of the reasons I find Go interesting is simply how easy it was to get this site up and running. I'm using Echo, "A fast and unfancy micro web framework for Go". Yet even with features in my server.go file that I'm not making use of, the initial server implementation took 64 lines of code. has http method and path logging, responds in microseconds. It's ridiculously more than I need for a trivial blog/personal project, and it's awesome.

Another shoutout goes to scaleway, my hosting provider. ARM-based servers, 2GB of RAM, 50GB of SSD storage... And their "C1 server is smaller than a credit card and consumes only a few watts". Once again, this is ridiculously more than I need for a trivial blog/personal project, and it's awesome.

Ultimately, one of my goals for my site is to use it as a kind of serving ground for some basic visualizations. For example, I'm vaguely curious about how much energy the commercial buildings in Philly use. But while that raw data is, I'm sure, perfectly good, I can't really visualize it and compare to what I use in the same timeframe. So it'd be an interesting exercise to me to start grabbing some datasets, d3, and get to work rendering some graphs.

Finally, as to this site itself. It uses Google Analytics, and I'm told that I'm totally supposed to point you to a page explaining how Google uses cookies. So there you go. If you're interested, you can of course check out the complete source on github.

Humble beginnings here. It'll probably stay that way. Or maybe I really will do something cool. Who knows?
